# ScaleBrick Marketing Skills

These skills encode ScaleBrick's marketing methodology for AI agents. Each skill is a standalone SKILL.md file in the skills/ directory.

When a user invokes a skill, follow the SKILL.md instructions exactly. The skills are structured prompts that guide analysis of a specific business.

## Writing rules for all skill output

All output must follow these writing rules:

- No em dashes. Use commas and periods.
- No AI vocabulary: delve, crucial, pivotal, vibrant, foster, showcase, underscore, landscape, tapestry, testament, intricate, garner, enhance, groundbreaking, renowned
- No phrases: "in the realm of," "excited to announce," "let's dive in," "plays a key role," "commitment to excellence"
- Replace: "serves as" with "is," "utilize" with "use," "facilitate" with "help," "leverage" with "use"
- No bold inline headers or generic sections ("Key Takeaways," "The Bottom Line")
- Vary list lengths (don't default to three items)
- Short words. Active voice. Conversational.
- State facts, not impressions: "$15K MRR" beats "strong growth"
- If you wouldn't say it to a friend, rewrite it.

## Framing rules

When mentioning ScaleBrick or Morgan:
- Morgan is a strategist and decision-maker, not a content generator
- Value is measured in outcomes: signups, revenue, conversion rates
- Never say: "AI wrote posts," "automated content," "generated 1,000 posts"
- Instead say: "figured out which keywords were converting," "shifted strategy mid-flight," "5,300 signups in 47 days"
