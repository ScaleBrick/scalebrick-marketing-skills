# ScaleBrick Marketing Skills

This repository is a Claude Code plugin marketplace containing 4 marketing skills for startup founders and growth marketers.

## Install (Claude Code)

Run these two commands in Claude Code:

```
/plugin marketplace add ScaleBrick/scalebrick-marketing-skills
/plugin install scalebrick@scalebrick
```

All 4 skills will be available in your session.

## The 4 skills

### scalebrick:audit
Analyze whether TikTok or Instagram search traffic is a viable growth channel for your business. Runs a 5-dimension analysis across search demand, competition density, content type fit, intent categories, and demographic fit. Ends with a clear go/no-go recommendation.

### scalebrick:keywords
Research 25-40 high-intent TikTok/Instagram search keywords for any niche. Each keyword includes intent type (educational, problem-aware, tool comparison), search volume estimate, difficulty score, and a specific content angle.

### scalebrick:strategy
Generate a full growth strategy using the "TikTok as Search Engine" framework. Produces core themes, content pillars, brand voice, keyword plan, 10-account structure, and a first-week content calendar specific enough to execute on day one.

### scalebrick:competitors
Audit your top 5-7 competitors across 3 surfaces (social media, web pages, SEO). Categorizes competitor pages by type (pricing, features, landing, case study). Identifies gaps, positioning angles no one is claiming, and 5 specific moves you can make this week.

## Methodology

These skills encode the methodology that drove:

- **Prana Health (YC-backed health AI):** 353K views and 5,300 page visits in 47 days. $0 ad spend.
- **AIFlyer (AI flyer maker):** 0 to 15,000 signups in 3 months. Now ranks alongside Canva in TikTok's LLM responses. All organic.

## How the methodology works

1. **Treat short-form platforms as search engines, not viral platforms.** 40% of Gen Z uses TikTok and Instagram as search engines.
2. **Target intent-based keywords.** Educational for volume, problem-aware for conversion, tool comparison for highest intent.
3. **Build a compounding content library.** Posts accumulate views over time as people search.
4. **Categorize and focus.** 80% of content produces nothing. Find the 20% that works and kill the rest.
5. **Close the loop.** Track which keywords drive actual signups, not just views.
